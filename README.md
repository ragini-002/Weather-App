# Weather App

A simple weather application that shows current weather information for any city.

## Live Demo

https://weather-app-mu-rose-65.vercel.app/

## Features

- Search weather by city name
- Display current temperature
- Show humidity
- Show wind speed
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript
- OpenWeather API
